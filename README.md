# ogre-vsbt-setup
A minimal setup for building [Ogre3D](https://www.ogre3d.org/) projects using [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022) from the command line.

```powershell
curl.exe -L -o "$(Get-Location)\codeDL.ps1" "https://raw.githubusercontent.com/Kiliken/ogre-vsbt-setup/main/codeDL.ps1"
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "$(Get-Location)\codeDL.ps1"

```
